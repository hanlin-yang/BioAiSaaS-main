# BioAiSaaS Frontend

## Lovable Project

The frontend application is built with Lovable and hosted at:
- **Project URL**: https://lovable.dev/projects/a53ebf3b-2c4a-464c-8ccd-aef27cfde3bd
- **GitHub Repository**: https://github.com/hanlin-yang/bioai-companion
- **Live Preview**: Available in Lovable project

## Features

- 7 Core Pages: Home, AI Console, Data Analysis, Tools Library, Knowledge Base, Projects, Settings
- Bilingual Support: Simplified Chinese / English
- Harvest Theme: Professional biomedical color scheme
- Responsive Design: Desktop and mobile support
- Tech Stack: React + TypeScript + Vite + Tailwind CSS + shadcn/ui

## Development

To run the frontend:

1. Access the Lovable project at the URL above
2. Use the live preview for testing
3. Or clone from bioai-companion repository (requires access)

## Integration with Backend

The frontend connects to the BioAiSaaS Python backend via REST API.

Backend API endpoints should be configured in the frontend environment variables.
